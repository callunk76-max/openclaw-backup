## SIRUP Bulukumba - Fitur Analisa Konsultansi

### Yang Dibangun
- **Backend API** (`/api/konsultan_check`): Matching logic antara paket Jasa Konsultansi (Perencanaan/Pengawasan) dengan Pekerjaan Konstruksi yang terkait
- **Frontend Modal**: Popup "Analisa Konsultansi" dengan card filter (Total/Overflow/Sedang/Aman) + tabel lengkap

### Matching Logic
- 5 level confidence: Pasti (100), Yakin (90), Sesuai (80), Mirip (70), Kemungkinan (60), Perkiraan (50+)
- Metode: Prefix cleanup, exact match, substring match, partial character match
- Filter by same satker + procurement type matching
- 60 pairs across 32 groups detected

### Fee Thresholds (Permen PUPR)
- Perencanaan: ≤3.5% dari pagu konstruksi
- Pengawasan: ≤2.5% dari pagu konstruksi
- Total: ≤6% (bukan 5%)
- Overflow >7%, Sedang 6-7%, Aman ≤6%

### Ambang Batas Metode Pemilihan (Perpres 46/2025)
- PL Konstruksi ≤Rp400jt (Ps.38)
- PL Jasa Konsultansi ≤Rp100jt
- PL Barang/Jasa Lainnya ≤Rp200jt
- Biaya Langsung Non Personel per Konsultan ≤40% (Perka LKPP No.9/2018)

### Z-index Stacking untuk Layered Modals
- Konsultansi: z-[70] (backdrop gelap)
- Detail Paket: z-[80] (tanpa backdrop)
- Detail Penyedia: z-[90] (tanpa backdrop)
- Saat close child modal, parent modal tetap kebuka (transisi smooth)

### Teknis

- Clickable cards untuk filter tabel

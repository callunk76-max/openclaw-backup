# 2026-05-02

## 📱 Build Fishing App - Mancing Bulukumba 🎣

Callunk requested a fishing spot app for **Kabupaten Bulukumba**, specifically for **Bira waters near Pulau Batu**.

### What was built:
- **PWA App** at `https://callunk.my.id/fish/` — Progressive Web App with GPS integration
- **APK** (native Android) — `App/fishing/Mancing Bulukumba.apk` (1.3MB, signed)
- **AAB bundle** — `App/fishing/Mancing Bulukumba.aab` for Play Store

### App features:
- 18 fishing spots (12 shore + 8 boat spots in Bira waters)
- GPS location detection & auto update
- BMKG weather/marine forecast for Bulukumba waters
- Tide tables, moon phase, solunar data
- Fish species guide (12 species)
- Interactive map with OpenStreetMap + Leaflet
- PWA offline capability (service worker + manifest)
- Boat navigation mode — tap marker → copy coords → navigate via GPS

### Key spots near Pulau Batu/Bira:
- Pulau Batu (-5.61800, 120.47200)
- Pulau Kambing (-5.63450, 120.46800)
- Pulau Liukang Loe (-5.66100, 120.49200)
- Gosong Karang Bira (-5.62500, 120.47500)
- Shark Point (-5.64100, 120.47300)
- Selat Bira-Bone (-5.66000, 120.52000)
- Karang Tengah Bira (-5.62800, 120.46000)

### Technical details:
- App served via nginx subpath `/fish/`
- Proxied to Python HTTP server on `127.0.0.1:5008`
- Permissions-Policy overridden to allow `geolocation=(self)` for the `/fish/` location
- APK generated via PWABuilder Cloud API (pwabuilder-cloudapk.azurewebsites.net)
- PWA metadata: `com.callunk.mancingblk` package, version 1.0.0.0

Template udah di-edit langsung di `templates/index.html`, tapi gunicorn nggak auto-reload template. Kalau ada perubahan template, **WAJIB restart gunicorn**:

gunicorn -w 4 -b 127.0.0.1:5005 --timeout 120 --access-logfile - --error-logfile - app:app </dev/null > /dev/null 2>&1 &

## SIRUP Bulukumba - Template Edits (Sesi 2026-05-05)

**KRITIKAL: Selalu restart gunicorn setelah edit template!** Gunicorn tidak auto-reload seperti Flask dev server. Jika lupa restart, user akan melihat versi lama.

### Perubahan yang diterapkan:

1. **Regulasi hanya di Tab 1 anomali**
   - Teks "Dasar regulasi ambang batas" dibungkus `<div id="anomaly-reg-info">`
   - Di `switchAnomalyTab()`: show/hide berdasarkan tab === 'pl'

2. **Fee table sub-rows default expanded**
   - Detail row tr: hapus `class="hidden"` → default kelihatan
   - Arrow ▶ mulai di `transform:rotate(90deg)` (kebuka)
   - `toggleFeeDetail()`: klik arrow collapse/expand

3. **Caption & Legend**
   - "Total Grup" → "Total"
   - Blok "🎯 Level keyakinan matching" dihapus

4. **Orphan detail z-index & blur**
   - `orphanModal` z-index: `z-[65]` → `z-[80]`
   - `openOrphanModal()`: tambah blur ke `anomalyModal` via class `modal-inactive`
   - `closeOrphanModal()`: hapus blur
   - CSS untuk `.modal-inactive` ada di awal `<style>` tag

### Pelajaran:
- Jangan edit template tanpa juga restart gunicorn
- Selalu cek keseimbangan `{ }` di JavaScript setelah edit
- Selalu verify via rendered HTML (curl + grep) bahwa perubahan teraplikasi
- Git backup ada (`cd /root/.openclaw/workspace/App/sirup_bulukumba && git log`)
- Jika ada major error, rollback dengan `git checkout -- templates/index.html` (HEAD adalah versi original stabil)

# 2026-05-11 Daily Log

## SIRUP Bulukumba App Fix
- **Issue**: Upload CSV error "Terjadi kesalahan sistem"
- **Root cause**: nginx config had `location ~ /(admin|wp-admin|...)` rule blocking ALL paths containing `/admin`, including the `/admin/upload` endpoint. nginx returned 444 before request reached Flask.
- **Fix**: Added `location ~ ^/admin/ { proxy_pass ... }` rule before the block rule, increased `client_max_body_size` to 110M and `proxy_read_timeout` to 120s for the root location serving the app.

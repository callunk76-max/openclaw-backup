### MT4 Key Learnings
- **OBJ_TREND** fails in some MT4 builds — use **OBJ_HLINE** instead for zone lines
- **OBJ_TEXT** with label style works for zone labels
- Historical scan in **OnInit()** when market closed; use **OnTick()** staggered for pair switching
- Panel objects must be minimized (8 objects) to avoid lag on pair switching
- `ObjectSetInteger(0, name, OBJPROP_COLOR, clrGreen)` works for OBJ_HLINE
- `ObjectSetText(labelName, text, size, font, color)` — no chart_id first param
- MQL4 doesn't have `VOLUME_TICK` constant — use `iVolume()` directly
- iATR() needs fallback when returns 0

## Key Architecture Decisions

### Currency Strength (GIRAIA) — `currency_strength.pine`
- Score per pair: 0-9 based on daily close position within daily range (`(close-low)/(high-low)`)
- Thresholds: ≥0.97→9, ≥0.90→8, ≥0.75→7, ≥0.60→6, ≥0.50→5, ≥0.40→4, ≥0.25→3, ≥0.10→2, ≥0.03→1, <0.03→0
- Currency strength = avg of 7 pairs (invert with `9-score` for quote currencies)
- Range: 0.0-9.0 per currency
- GAP = base_strength - quote_strength (range -9 to +9, display absolute value with color)

### Hybrid Scoring (Opsi A+B merge)
| Component | Weight | Default ON? |
|-----------|--------|-------------|
| GAP ≥ 5 | Gatekeeper | ✅ Always |
| GT (EMA Gates) | 30% (+1~+3) | ✅ Always |
| BB+RSI | 20% (+1) | ❌ Toggle |
| VOL (ATR regime) | 15% (±1) | ❌ Toggle |
| SnR | 10% (+1) | ❌ Toggle |
| BB standalone | (+1) | ❌ Toggle |
| GAP≥7 | Bonus (+2) | ✅ Always |
| Conflict | Penalty (−2) | ✅ Always |

### Signal Hysteresis
- STRONG BUY(2) → SELL(-1/-2): blocked, goes through NEUTRAL(0) first
- STRONG SELL(-2) → BUY(1/2): blocked, goes through NEUTRAL first
- STRONG(2) → NEUTRAL(0): steps down to BUY(1) first before neutral
- STRONG(-2) → NEUTRAL(0): steps up to SELL(-1) first

### Scoring → Signal Mapping
- Score ≥ 7 → STRONG BUY/SELL
- Score ≥ 4 → BUY/SELL
- Score < 4 → WAIT/NEUTRAL
- Score 3-4 range is "weak" - signal may exist but barely

### F+/F- Buttons
- ResizeFont() deletes all objects and recreates dashboard from scratch
- `useLH = (int)(fs * 1.8 + 1)` — line height auto-calculates
- `colScale = fs / 8.0` — column positions scale proportionally
- Bottom bar (MAX, BAL, MAR, F+/-, Auto, Alert) also scales with `cs`

### Trailing Stop
- Activation: profit ≥ ATR × 1.0
- Trail SL: peak - ATR × 0.8 (for BUY)
- Update: only when distance > ATR × 0.2
- Auto-cleanup: removes closed positions from tracking each tick
- Broker guard: checks MODE_STOPLEVEL before modify

## Bug Fixes
- `OnTick()` → `OnCalculate()` for indicators (MQL4 restriction)
- `OBJPROP_RAY_RIGHT` → `OBJ_HLINE` + delete RAY prop
